# Sakurai's Revenge

Unity Team Project for Game Programming 1402

## Mechanics:
Use your stealth skills to clear the island of enemies.

## Controls:
Left-Mouse Button: Shuriken Throw  
Right-Mouse Button: Stab  
Left Shift: Sprint  
Left Ctrl: Crouch  
Spacebar: Jump
